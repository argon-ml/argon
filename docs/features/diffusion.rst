Diffusion Processes
===================

.. currentmodule:: foundry.diffusion

The :py:mod:`foundry.diffusion` module contains schedules for
different diffusion processes. For instance a :py:class:`DDPMSchedule`
can be used to build and train a DDPM diffusion process.
