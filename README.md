# Installation Instructions

 1. Install [uv](https://github.com/astral-sh/uv)
    ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ```
 2. To set up the argon environment, run
    ```bash
    uv sync
    ```
 3. Then try some notebooks:
    ```bash
    uv run ...
    ```
